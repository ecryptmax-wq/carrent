CarRent

Modern Car Rental Landing Page with Google Forms Integration

Version 1.0 | November 10, 2025

Project Overview

CarRent is a fully responsive, single-page car rental website built with HTML, CSS, and JavaScript. It features:

  • Interactive booking form with real-time vehicle selection
  • Checkout modal with pre-filled trip details
  • Google Forms backend for collecting booking data
  • Leaflet.js map, Font Awesome icons, and Toastr notifications
  • No server required — 100% static

Customizing Images

1. Car Images in Dropdown

Each car option uses data-img to show preview in modal:

<option value="BMW 3 Series" data-img="assets/img/bmw.jpg">BMW 3 Series</option>

Steps:

  1. Place your image in assets/img/
  2. Update data-img="assets/img/your-car.jpg"
  3. Use 800x600px JPG/PNG for best quality

2. Vehicle Cards

Replace Unsplash images:

<img src="assets/img/toyota-camry.jpg" alt="Toyota Camry">

Google Forms Integration

Warning: You must create a Google Form with exact field order and types.

Step 1: Create Google Form

  1. Go to forms.google.com
  2. Create Blank Form
  3. Title: Your Title
  4. Ensure the form questions are added in this exact order:

Step 2: Add Questions (Exact Order)

Step 3: Get entry.xxx IDs

  1. Click Send → </> (Embed)
  2. Or open form in preview → Right-click → Inspect
  3. Find inputs like:
  4. Make sure each entry.xxx field matches the corresponding form column exactly and is not mixed up.
<input name="entry.1190199788" ...>

Step 4: Get Form Action URL

From embed code:

https://docs.google.com/forms/d/e/YOUR_FORM_ID/formResponse

Step 5: Update Script in index.htmlAdd this script before </body>:

<script>
document.getElementById("checkout-form").addEventListener("submit", function(e) {
  e.preventDefault();

  // Copy booking data to hidden fields
  document.getElementById("pickup-date").value = document.getElementById("pick-up-date-ph").textContent.trim();
  document.getElementById("pickup-time").value = document.getElementById("pick-up-time-ph").textContent.trim();
  document.getElementById("pickup-location").value = document.getElementById("pickup-location-ph").textContent.trim();
  document.getElementById("selected-car").value = document.getElementById("selected-car-ph").textContent.trim();

  const formData = new FormData();
  formData.append("entry.1190199788", document.getElementById("firstName").value);
  formData.append("entry.2013771470", document.getElementById("lastName").value);
  formData.append("entry.455831811", document.getElementById("phone").value);
  formData.append("entry.1600546778", document.getElementById("age").value);
  formData.append("entry.1170856035", document.getElementById("zip").value);
  formData.append("entry.1950349338", document.getElementById("email").value);
  formData.append("entry.1140226880", document.getElementById("address").value);
  formData.append("entry.1148995167", document.getElementById("city").value);
  formData.append("entry.755968688", document.getElementById("pickup-location").value);
  formData.append("entry.1394724332", document.getElementById("pickup-date").value);
  formData.append("entry.1589860892", document.getElementById("pickup-time").value);
  formData.append("entry.1049012762", document.getElementById("selected-car").value);

  fetch("https://docs.google.com/forms/d/e/YOUR_FORM_ID_HERE/formResponse", { 
    method: "POST",
    mode: "no-cors",
    body: formData
  }).then(() => {
    alert("Booking confirmed! We'll contact you soon.");
    document.getElementById("checkoutModal").classList.remove("show");
  }).catch(() => {
    alert("Booking submitted!");
  });
});
</script>
Success: Replace YOUR_FORM_ID_HERE and all entry.xxx with your real values.

Booking Flow

  1. User selects location, date, time, car → clicks Book Now
  2. Modal opens with pre-filled trip details
  3. User enters personal info
  4. JS copies hidden values (date, car, etc.)
  5. Form submits to Google Forms via fetch()
  6. Success message → modal closes

Testing the Form

  1. Fill booking form → click "Book Now"
  2. Complete checkout modal
  3. Click "Confirm Booking"
  4. Check Google Sheets (Responses tab)
